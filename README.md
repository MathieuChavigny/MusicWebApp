Développement d’un site web en JavaScript REACT, ce site permet à l’utilisateur de créer un compte et de sauvegarder plusieurs listes de lecture de ses musiques préférées en plus de pouvoir écouter des extraits de celles-ci. Aussi, il peut rechercher à peu près n’importe quel artiste, album ou chanson grâce à la barre de recherche. Le tout étant jumelé avec une base de donnée dans firebase.
